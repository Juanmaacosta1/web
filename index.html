Características del Sistema

Base de Datos PostgreSQL

Sistema de base de datos robusto y confiable con pgAdmin para gestión visual.

Desarrollado con Python/Flask

Backend en Python con Flask Framework, frontend con Bootstrap y JavaScript.

Gestión Completa de Caja

Apertura/cierre de turnos, control de montos iniciales y finales, diferencia calculada.

Pagos Combinados

Acepta múltiples métodos de pago en una sola venta (efectivo, tarjeta, transferencia).

Tickets Personalizados

Generación de tickets profesionales con todos los datos de la venta.

Reportes Detallados

Ventas por fecha, productos más vendidos, análisis de rentabilidad.
